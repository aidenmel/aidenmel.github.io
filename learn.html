      Creating Scroll Effects

      My portfolio is centered around interactions. On this page, you'll learn how to create and integrate a basic scroll effeect for your website.

      Continue scrolling to move forwards.

      What are Scroll Effects?

      Scroll effects are animations or manipulations that occur when the client reaches ("scrolls pass") a specific point on the page. Most effects are integrated with javascript, though some can be made with purely css.

      How does it work?

      This works by comparing the object's absolute position to that of the cilent's active position (the position at which the client has scrolled). So, if the scroll position is greater than the element's absolute position, show the object. Else, hide the object.

      Make Effects Appear Sooner

      Currently, the effect will only occur when the object is at the tippy-top of the page. Therefore, to make the effect take place near the bottom of the page, we will have to account for the window's height.

      To accomplish this, divide $(window).height() by 3 and multiply that result by 2. Then, subtract the value from the object's ABSOLUTE_POSITION.

      ...
    // .each() loops through all objects with the ".fade-fx" class
    $(".fade-fx").each(function(){

        // Returns the absolute top position
        var ABSOLUTE_POSITION = $(this).offset().top;
        
        
      //  Subtracts half the screen's width from ABSOLUTE_POSITION
        ABSOLUTE_POSITION -= (2 * ($(window).height() / 3));
      
    })

      Incorporating CSS

      With javascript set aside, lets now focus on using CSS to create a actual effect.

      Create a Effect (1/2)

      Hide Objects

      With javascript set aside, lets now focus on using CSS to create a actual effect.

      First, use .fade-fx to hide the object. This can done by setting the opacity to 0. Then, add even more properties like blur to make the effect pop out.

      .fade-fx {
        opacity: 0; /* Hides the object */
        filter: blur(16px); /* Makes the object blurry */
        transition: all 250ms ease-out; /* Makes a transition/animation */
    }

      styles.css

      Create a Effect (2/2)

      Activating Objects

      Remember: The active class is added to each element that is in frame.

      With this, use .active to revert the effect.

      .fade-fx {
    opacity: 0; /* Hides the object */
    filter: blur(16px); /* Makes the object blurry */
    transition: all 250ms ease-out; /* Makes a transition/animation */
}


      .fade-fx.active {
    opacity: 1; /* Shows the object */
    filter: blur(0); /* Removes the blur effect */
}

      styles.css

      Tips & Tricks

      Did you know?

      You successfully created a basic effect, but you can do more. Try incorporating the use of IDs to make item-specific animations. So instead of a generic animation across all object, use ids to make one object appear and another to rotate.
